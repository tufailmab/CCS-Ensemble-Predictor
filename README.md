<h1>CCS Ensemble Predictor</h1>

<p>
This repository contains the standalone application, trained models, and source code developed for the study:
</p>

<p>
<strong>“Development of a Standalone Application for Accurate and User-Friendly Prediction of Concrete Compressive Strength Using Ensemble Machine Learning.”</strong>
</p>

<p>
The project implements a diversified stacked ensemble of machine learning models to accurately predict concrete compressive strength using the UCI benchmark dataset. It also includes cross-model feature analysis to provide interpretable insights for practical mix design.
</p>
